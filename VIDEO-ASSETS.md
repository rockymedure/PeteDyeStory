## How to Use

### Finding Specific Moments

1. **Read the AI summary first:**
   ```
   video-processing/output/[Video_Name]/analysis/simple_director_analysis.md
   ```

2. **Search full transcript for keywords:**
   ```bash
   grep -i "pete dye" video-processing/output/*/analysis/full_transcript.txt
   ```

3. **Use pre-extracted clips:**
   ```
   video-processing/output/[Video_Name]/clips/
   ```

### Quick Search Commands

```bash
# Find all mentions of Pete across transcripts
grep -ri "pete" video-processing/output/*/analysis/full_transcript.txt

# Find bell mentions (for #12 moment)
grep -ri "bell" video-processing/output/*/analysis/full_transcript.txt

# Find opening/grand opening mentions
grep -ri "opening\|grand" video-processing/output/*/analysis/full_transcript.txt

# List all clips by size (largest first)
find video-processing/output -name "*.mp4" -path "*/clips/*" -exec ls -lh {} \; | sort -k5 -hr
```
